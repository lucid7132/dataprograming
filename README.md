데이터분석프로그래밍 1조

# 1차 전처리 작물 파트 - 준영
Processing/Crop 폴더 내에 존재하는 4개의 파이썬 파일

    main.py : 최종 로직 실행 함수. 아래 3개의 파이썬 파일을 실행한 뒤 출력된 csv 결과 데이터를 사용하여 3개의 그래프를 그림.
    py_mix.py : 아래 2개의 파이썬 파일을 실행한 뒤 출력된 csv 결과 데이터를 사용하여 두 데이터를 하나의 csv 파일로 병합.
    py_price_at.py : at도매시장 통합 홈페이지 데이터를 사용해 5개의 주요 작물에 대한 데이터를 전처리
    py_price_nongnet.py : 농넷사이트를 사용한 5개의 주요 작물에 대한 데이터를 전처리
    
결과 - 각 지역별 주요 작물에 대한 1kg당 면적 대비 가격은 변동성이 크지만 이는 타 지역도 비슷한 양상을 띰. 

    10a당 품목별 생상량은 남부지방 양파와 강원지방 무, 그 다음으로 강원지방 배추가 생산량이 높음.
    반면 상대적으로 생산량이 적은 사과와 마늘이지만 1kg당 가격을 환산하면 사과가 가장 소득이 높고 마늘도 생산량 대비 뒤처지지 않음.
    이에 증명하듯 경상북도(사과 주 생산지) 타 지역 대비 귀농인구수가 높은게 보임.
